# 共通ルール
- 出力は日本語
- コードにはコメントをつける
- まとめはMarkdown形式
- 新しい調査メモは Templates/ のテンプレートを使う
- Vault/ はObsidianで管理する（Claude Codeでは編集しない）

## プロジェクト一覧
- Projects/AI_Skills/  … AIツールのSkills・ベストプラクティス収集
- Projects/UE5/        … UE5ソースコード解析
- Projects/GameTech/   … ゲーム技術全般
- Projects/GameDesign/ … ゲームデザイン理論・パターン
- Projects/Art/        … イラスト・3Dモデリング

## トークン節約ルール
- 作業開始時に対象プロジェクトの CLAUDE.md を必ず参照する
- 長い調査は要点のみ返答し、詳細はファイルに書き出す
